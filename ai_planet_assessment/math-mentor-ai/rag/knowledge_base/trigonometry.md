# Trigonometry - JEE Mathematics Knowledge Base

## Basic Definitions

### In Right Triangle
- sin θ = Opposite/Hypotenuse
- cos θ = Adjacent/Hypotenuse
- tan θ = Opposite/Adjacent = sin θ/cos θ
- cot θ = Adjacent/Opposite = cos θ/sin θ = 1/tan θ
- sec θ = Hypotenuse/Adjacent = 1/cos θ
- csc θ = Hypotenuse/Opposite = 1/sin θ

### Unit Circle Definition
For point P(x, y) on unit circle at angle θ from positive x-axis:
- cos θ = x
- sin θ = y

---

## Fundamental Identities

### Pythagorean Identities
- sin²θ + cos²θ = 1
- 1 + tan²θ = sec²θ
- 1 + cot²θ = csc²θ

### Reciprocal Identities
- sin θ × csc θ = 1
- cos θ × sec θ = 1
- tan θ × cot θ = 1

---

## Compound Angle Formulas

### Sum Formulas
- sin(A + B) = sin A cos B + cos A sin B
- sin(A - B) = sin A cos B - cos A sin B
- cos(A + B) = cos A cos B - sin A sin B
- cos(A - B) = cos A cos B + sin A sin B
- tan(A + B) = (tan A + tan B)/(1 - tan A tan B)
- tan(A - B) = (tan A - tan B)/(1 + tan A tan B)

### Double Angle Formulas
- sin 2A = 2 sin A cos A = 2 tan A/(1 + tan²A)
- cos 2A = cos²A - sin²A = 2cos²A - 1 = 1 - 2sin²A = (1 - tan²A)/(1 + tan²A)
- tan 2A = 2 tan A/(1 - tan²A)

### Triple Angle Formulas
- sin 3A = 3 sin A - 4 sin³A
- cos 3A = 4 cos³A - 3 cos A
- tan 3A = (3 tan A - tan³A)/(1 - 3 tan²A)

### Half Angle Formulas
- sin(A/2) = ±√[(1 - cos A)/2]
- cos(A/2) = ±√[(1 + cos A)/2]
- tan(A/2) = (1 - cos A)/sin A = sin A/(1 + cos A) = ±√[(1 - cos A)/(1 + cos A)]

---

## Product-to-Sum Formulas

- 2 sin A cos B = sin(A + B) + sin(A - B)
- 2 cos A sin B = sin(A + B) - sin(A - B)
- 2 cos A cos B = cos(A + B) + cos(A - B)
- 2 sin A sin B = cos(A - B) - cos(A + B)

---

## Sum-to-Product Formulas

- sin C + sin D = 2 sin[(C + D)/2] cos[(C - D)/2]
- sin C - sin D = 2 cos[(C + D)/2] sin[(C - D)/2]
- cos C + cos D = 2 cos[(C + D)/2] cos[(C - D)/2]
- cos C - cos D = -2 sin[(C + D)/2] sin[(C - D)/2]

---

## Allied Angles

### Quadrant Rules (ASTC - All Students Take Coffee)
- Q1 (0° to 90°): All positive
- Q2 (90° to 180°): Only sin, csc positive
- Q3 (180° to 270°): Only tan, cot positive
- Q4 (270° to 360°): Only cos, sec positive

### Standard Transformations
- sin(90° - θ) = cos θ, cos(90° - θ) = sin θ
- sin(90° + θ) = cos θ, cos(90° + θ) = -sin θ
- sin(180° - θ) = sin θ, cos(180° - θ) = -cos θ
- sin(180° + θ) = -sin θ, cos(180° + θ) = -cos θ
- sin(-θ) = -sin θ, cos(-θ) = cos θ (odd and even functions)

## Inverse Trigonometric Functions

### Principal Value Ranges
- sin⁻¹x ∈ [-π/2, π/2], domain: [-1, 1]
- cos⁻¹x ∈ [0, π], domain: [-1, 1]
- tan⁻¹x ∈ (-π/2, π/2), domain: ℝ
- cot⁻¹x ∈ (0, π), domain: ℝ
- sec⁻¹x ∈ [0, π] - {π/2}, domain: |x| ≥ 1
- csc⁻¹x ∈ [-π/2, π/2] - {0}, domain: |x| ≥ 1

### Important Identities
- sin⁻¹x + cos⁻¹x = π/2
- tan⁻¹x + cot⁻¹x = π/2
- sec⁻¹x + csc⁻¹x = π/2
- tan⁻¹x + tan⁻¹y = tan⁻¹[(x + y)/(1 - xy)] when xy < 1
- tan⁻¹x - tan⁻¹y = tan⁻¹[(x - y)/(1 + xy)] when xy > -1
- 2tan⁻¹x = sin⁻¹[2x/(1 + x²)] = cos⁻¹[(1 - x²)/(1 + x²)] = tan⁻¹[2x/(1 - x²)]

## Properties of Triangles

### Sine Rule
a/sin A = b/sin B = c/sin C = 2R
where R is circumradius

### Cosine Rule
- a² = b² + c² - 2bc cos A
- b² = a² + c² - 2ac cos B
- c² = a² + b² - 2ab cos C

### Projection Formula
- a = b cos C + c cos B
- b = c cos A + a cos C
- c = a cos B + b cos A

### Area of Triangle
- Area = (1/2)ab sin C = (1/2)bc sin A = (1/2)ca sin B
- Area = √[s(s-a)(s-b)(s-c)] (Heron's formula)
where s = (a + b + c)/2

### Half Angle Formulas for Triangle
- sin(A/2) = √[(s-b)(s-c)/(bc)]
- cos(A/2) = √[s(s-a)/(bc)]
- tan(A/2) = √[(s-b)(s-c)/(s(s-a))]

### In-radius (r) and Circumradius (R)
- r = Area/s = (s-a)tan(A/2)
- R = a/(2 sin A) = abc/(4 × Area)

## Heights and Distances

### Angle of Elevation
Angle made with horizontal when looking UP at an object

### Angle of Depression
Angle made with horizontal when looking DOWN at an object

### Important Formulas
- Height = Distance × tan(angle of elevation)
- For two angles α and β from same point to object of height h:
  Distance = h(cot α - cot β)

## JEE-Specific Tips for Trigonometry

1. **For compound angles**: Convert everything to sin and cos when stuck
2. **For equations**: Use factorization and standard substitutions
3. **For inverse trig**: Always check the principal value range
4. **For triangle problems**: Draw diagram, apply sine/cosine rule appropriately
5. **For heights and distances**: Identify the right triangles first
6. **For proving identities**: Usually work on the more complex side
